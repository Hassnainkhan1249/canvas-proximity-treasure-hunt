# Canvas Proximity Treasure Hunt

Canvas Proximity Treasure Hunt is an interactive browser-based game built with React, TypeScript, and Vite.

## Features

- Interactive canvas-based gameplay
- Proximity-based treasure hunting
- Clean and responsive interface
- Built using modern frontend technologies

## Technologies Used

- React
- TypeScript
- Vite
- CSS
